# Sage IntegralLattice Reference

Agent-facing reference material for using and inspecting Sage's `IntegralLattice`
surface while working on lattice computations.

## Files

- `quick-reference.md`: common `IntegralLattice` methods and usage traps.
- `method-discovery.md`: source and method-discovery patterns for Sage classes.
- `source-files.md`: old local Sage source-file map retained as provenance; verify paths
  before relying on them.

These notes are operational guidance. Mathematical summaries and paper findings belong
under `notes/`.
