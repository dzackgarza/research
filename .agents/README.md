# Agent Notes

Agent-facing material for this umbrella repository lives here.

Use this tree for tool usage notes, source-discovery guidance, workflow references, and
agent provenance that help agents operate the research workspace. Human mathematical
summaries and research findings belong under `notes/`.

## Layout

- `references/`: reusable operational references for agents.
- `provenance/`: imported agent instructions retained to explain historical agent
  behavior or migration decisions.

Do not place paper summaries, mathematical findings, or live project research notes in
this tree.
