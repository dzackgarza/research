# Enriques Moduli Computations

Computational specifications and stubs for Enriques surface orbit computations.

## Files

- `ENRIQUES_ORBIT_COMPUTATION_SPEC.md`: mathematical and implementation specification for Enriques orbit computations.

- `SAGEMATH_TABLE_REPRODUCTION_SPEC.md`: Sage interface requirements for reproducing table calculations.

- `SAGEMATH_WRAPPER_SPEC.md`: wrapper design notes for a Sage-facing workflow.

- `enriques_sage_stub.py`: imported Sage stub with test cases from the old work tree.

General Dawes orbit computations live in `computations/lattice-orbits/`.
